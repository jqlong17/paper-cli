# 特殊字符测试

## HTML 特殊字符

- &lt; 小于号
- &gt; 大于号
- &amp; 与号
- &quot; 引号
- &#39; 单引号

## Markdown 特殊字符

- *星号*
- _下划线_
- `反引号`
- [方括号]
- (圆括号)
- {花括号}
- <尖括号>

## 转义字符

\* 不是斜体
\` 不是代码
\[ 不是链接
\< 不是 HTML
